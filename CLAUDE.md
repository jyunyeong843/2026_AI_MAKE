# 복지온길 (Bokji-On-Gil)

어르신 친화 보행 경로 안내 서비스. GPS 기반 현재 위치에서 어르신이 자주 방문하는 공공·복지 시설까지, 경사·계단·강수량을 고려한 우회 경로를 추천한다.

## 1. 화면

### 첫 번째 화면 (도착지 설정)
- 2행 2열로 배치된 사각형 버튼들로 도착지 목록을 표시한다.
- 각 사각형 버튼을 클릭하면 두 번째 화면(지도)으로 넘어간다.
- 예시 도착지: 성북구청, 동선동 주민센터, 경로당, 삼선동 주민센터.

### 두 번째 화면 (지도)
- 현재 GPS 위치를 지도 화면에 띄운다.
- 첫 번째 화면에서 선택한 도착지까지의 우회 경로를 표시한다.
- 현재 위치에서 도착 위치까지의 경로 결정 규칙은 아래 **2. 지도 룰**을 따른다.
- 하단에 경로 유형(예: "경사로 우회경로")과 예상 소요 시간(예: "15분")을 표시한다.

## 2. 지도 룰 (경로 결정 규칙)

1. 사용자의 GPS 위치를 기준으로 반경 1km 이내의 공공기관 목록을 출력한다.
   - 대상 시설: 주민센터, 동사무소, 구청, 복지관, 문화센터 등 어르신이 방문할 가능성이 높은 공공·복지 시설로 한정한다.
2. 사용자가 목적지를 선택하면 **네이버 지도 API**를 통해 기본 최단 경로를 탐색한다.
3. 탐색된 경로를 구간별로 나누어 각 구간의 **경사도, 계단 여부, 현재 시간당 강수량**을 분석한다.
4. **평상시 (시간당 강수량 10mm 미만)**
   - 경사도 5% 미만 구간: 정상 통과 처리
   - 경사도 5~10% 구간: 약한 가중치 부여
   - 경사도 10% 초과 구간 및 계단 구간: 강한 가중치 부여
5. **우천 시 (시간당 강수량 10mm 이상)**
   - 경사도 5% 미만 구간: 정상 통과 처리
   - 경사도 5~10% 구간: 강한 가중치 부여
   - 경사도 10% 초과 구간 및 계단 구간: 통행 불가 구간으로 판단하여 경로 탐색 대상에서 제외
6. AI는 각 경로의 **총 이동 거리, 예상 시간, 경사 가중치, 우천 가중치**를 종합하여 최종 보행 비용을 계산한다.
7. 최종 보행 비용이 가장 낮은 경로를 **어르신 친화 우회 경로**로 추천한다.

## 3. TTS

- 두 번째 화면이 시작되면서 **"목적지까지의 경로안내를 시작합니다"** 를 TTS로 출력한다.
